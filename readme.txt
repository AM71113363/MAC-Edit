run MAC-Edit.exe (32Bit Windows)

1.Select the device (it works only on Wifi Cards or USB wifi dongle).
2.Click [ info ] to read the actual MAC address.
3.Click [ change ] to automatically change MAC address or you can change manually the values.
4.When done the Program will exit.(about 4 seconds).



IMPORTANT:
Remember the actual MAC address, there is no backup feature with "MAC-Edit.exe",
